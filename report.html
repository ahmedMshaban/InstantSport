Introduction:

InstantSport is the place for on-demand sports news, and we wanted to make sports news easy and fun even for casual sports lovers. All sports in one place.

State 3 things that have inspired you when creating your website:

  • Stories from the fox sports website.
  • Sticky sidebar of the playmakers website.
  • Good color choice of MARCA website.

State 3 ways in which your site is accessible:

  • Image elements have [alt] attributes
  • All pages has a (title) element
  • List items (li) are contained within (ul) or (ol) parent elements

State 3 ways in which you have considered usability in your site:

  • Sticky sidebar which helps users have a more pleasant, seamless experience.
  • The main menu doesn’t have too high a number of submenus which would help quickly give users what they want.
  • The design and layout are simple, the space between sections is constant, and using propr images to attract users.

State 3 things you had to learn/find out by yourself when creating your site, and describe how you did that:

  • I learned how to use CSS grid, CSS tricks have a comprehensive article.
  • I learned how to make my more accessible by using the Google Lighthouse tool.
  • I learned how to use graphic tools such as Canava to create the site logo and edit images.

What aspect(s) of your work do you think worked well and why?

  • I tried as possible to follow the best practices I learned throughout the course to build a clean and zero error website as possible and I have used Google Lighthouse best practice tool to help me with that.
  • I have used w3c to scan and validate the website HTML, CSS and make sure it’s error-free.
  • The design, style, and responsiveness I think those work very well and the site is adept very well on all devices.

What aspect(s) of your work could be improved, and how might you do things differently another time?

  • I would make the home page slider move automatically every 5 seconds, now you have to manually scroll left and right.
  • I would implement an infinite scroll on the “All news” sections.
  • Add back to the top button after implementing the infinite scroll to make the site more accessible.

What resources were used when creating your work?

  • Canva to design logo and edit the images.
  • Google fonts to load the fonts.
  • W3C markup validator, W3C CSS validator
  • Google Lighthouse to scan and improve the performance.
  • Pexels to get stock photos and I referred the authors as needed.
  • Favicon & App Icon Generator
  • SVG PRO to get the SVG icons.
  • Figma to design the mock and wireframe.
  • CSS reset to reduce browser inconsistencies.
  • Taleek used to download mobile app images
